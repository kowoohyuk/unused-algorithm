## 파도반 수열

> 출처 : <https://www.acmicpc.net/problem/9461>

### 답안

```python
T = int(input())
for _ in range(T):
    n = int(input())
    lines = [0, 1, 1, 1, 2] + ([0] * (n - 4))
    if n <= 4:
        print(lines[n])
        continue
    for i in range(5, n + 1):
        lines[i] = lines[i - 1] + lines[i - 5]
    print(lines[n])


```
