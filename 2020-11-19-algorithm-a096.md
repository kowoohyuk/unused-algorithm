## JadenCase 문자열 만들기

> 출처 : <https://programmers.co.kr/learn/courses/30/lessons/12951?language=javascript>

### 답안

```javascript
function solution(s) {
    const answer = [];
    s = s.toLowerCase();
    for(let i = 0; i < s.length; i++) {
        if(i == 0 || s[i - 1] === ' ' && s[i] !== ' ') {
            answer.push(s[i].toUpperCase());
            continue;
        }
        answer.push(s[i]);
    }
    return answer.join('');
}

```
