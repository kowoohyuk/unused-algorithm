## 셀프 넘버

> 출처 : <https://www.acmicpc.net/problem/4673>

### 답안

```javascript
function selfNumber(n) {
    nums_map = new Set();
    result = '';
    for(let i = 1; i < n; i++) {
        nums_map.add(i + ('' + i).split('').reduce((acc, v) => acc += +v, 0));
    }
    for(let i = 1; i < n; i++) {
        if(!nums_map.has(i)) result += i + '\n';
    }
    console.log(result);
}
selfNumber(10000);

```
