## daily temperatures

> 출처 : <https://leetcode.com/problems/daily-temperatures/>

### 답안

```python
class Solution:
    def dailyTemperatures(self, T: List[int]) -> List[int]:
        stack = []
        result = [0] * len(T)
        for i in range(len(T)):
            cur = T[i]
            while stack and T[stack[-1]] < cur:
                last = stack.pop()
                result[last] = (i - last)
            stack.append(i)
        return result
```
