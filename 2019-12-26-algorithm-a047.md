## 가장 큰 수

> 출처 : <https://programmers.co.kr/learn/courses/30/lessons/42746?language=javascript>

### 답안

```javascript
function solution(numbers) {
  const result = +numbers.join('') < 1 ? 0 
    :numbers.sort((a,b) => ''+a+b > ''+b+a ? 1 : -1).reverse().join('');
  return numbers[0] ? result : '0';
}
```
