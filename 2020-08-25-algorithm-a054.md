## 짝지어 제거하기

> 출처 : <https://programmers.co.kr/learn/courses/30/lessons/12973?language=javascript>

### 답안

```javascript
function solution(s) {
  let stack = [];
  for (let i = 0; i < s.length; i++) {
    if (stack[stack.length - 1] === s[i]) {
      stack.pop();
    } else {
      stack.push(s[i]);
    }
  }
  return stack.length ? 0 : 1;
}
```
