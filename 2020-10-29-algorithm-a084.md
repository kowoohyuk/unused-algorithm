## 쉬운 계단 수

> 출처 : <https://www.acmicpc.net/problem/10844>

### 답안

```python
n = int(input())
d = [[0 for y in range(10)] for x in range(n + 1)]
for y in range(1, 10):
    d[1][y] = 1
for x in range(2, n + 1):
    for y in range(10):
        if y == 0:
            d[x][0] = d[x - 1][1]
        elif y == 9:
            d[x][9] = d[x - 1][8]
        else:
            d[x][y] = (d[x - 1][y - 1] + d[x - 1][y + 1]) % 1000000000
print(sum(d[n]) % 1000000000)
```
