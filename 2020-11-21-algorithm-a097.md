## 숫자의 표현

> 출처 : <https://programmers.co.kr/learn/courses/30/lessons/12924?language=python3>

### 답안

```python
def solution(n):
    answer = 1
    for i in range(1, n):
        sum = i
        j = i + 1
        while(sum < n):
            sum += j
            j += 1
        if sum == n:
            answer += 1            
    return answer

```
