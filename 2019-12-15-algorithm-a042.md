## 쇠막대기

> 출처 : <https://programmers.co.kr/learn/courses/30/lessons/42585?language=javascript>

### 답안

```javascript
function solution(s) {
    let result = 0, count = 0;
    for(let i = 0; i < s.length; i += 1) {
        if(s[i] == '(' && s[i + 1] == ')') { result += count;
          i += 1;
        } else if(s[i] == '(') {
            count += 1;
        } else if(s[i] == ')') {
            count -= 1;
            result += 1;
        }
    }
    return result;
}
```
