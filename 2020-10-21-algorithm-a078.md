## ACM 호텔

> 출처 : <https://www.acmicpc.net/problem/10250>

### 답안

```python
try_num = int(input())

for i in range(try_num):
    H, W, P = map(int, input().split())
    if P % H == 0:
        floor = H
        Num = int(P/H)
    else:
        floor = P % H
        Num = int(P/H) + 1
    if Num >= 10:
        print(str(floor) + str(Num))
    else:
        print(str(floor) + '0' + str(Num))
```
